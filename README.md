# py-rtsp-recorder
A simple rtsp recorder

## Installation

### Install ffmpeg

On debian/ubuntu :
```shell
$ sudo apt install -y ffmpeg
```

### Install python

On debian/ubuntu :
```shell
$ sudo apt install -y python3-full
```

### Install dependencies
```shell
$ pip3 install -r requirements.txt
```

### Configure the project
copy .env.exemple to .env

Edit values with yours

## Usage
```shell
$ python3 recorder.py
```
